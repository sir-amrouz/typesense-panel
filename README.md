# Typesense Panel (typesense-panel)
A Typesense panel to manage multi instances built with angular 21
## Web
> [!IMPORTANT]
> **HTTPS Requirement.**
> If your client application is served via HTTPS, your backend server must also be configured with SSL/TLS. Requests to non-secure endpoints will fail due to **Mixed Content** errors.
>
> *Do not rely on browser overrides to allow mixed content, as this poses significant security risks.*
